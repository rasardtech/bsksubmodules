Simplified Decimal Display

Intelligent Decimal Precision Formatting for Odoo

Transform how your Odoo instance displays numbers. This module introduces a smart formatting system that automatically adjusts decimal precision based on the context of the value—quantitative or monetary.

Quantitative Values

Automatically strips all trailing zeros for cleaner display.

Example: 10.12300 → 10.123
Example: 10.12000 → 10.12
Example: 10.10000 → 10.1
Example: 10.00000 → 10

Monetary Values

Maintains a minimum of 2 decimal places for financial accuracy.

Example: $10.12300 → $10.123
Example: $10.12000 → $10.12
Example: $10.10000 → $10.10
Example: $10.00000 → $10.00

IMPORTANT

If you uninstall this module, you must restart the Odoo server to revert the decimal formatting changes correctly.

Support & Configuration

This module works out-of-the-box with support for both dot and comma decimal separators. No complex configuration required.
