Cleaner Expense Management

Fix the default "Reset to Draft" behavior and keep your Journal Entries clean.

Expense Reset Fix improves the accounting workflow within the Odoo Expenses app. It prevents the creation of unnecessary "Reverse Entries" when resetting an expense report and ensures linked Journal Entries are properly deleted when the report is removed.

The Problem

Why do you need this module?

In the standard Odoo configuration, handling approved expense reports can result in cluttered accounting records:

  • Reverse Entry Clutter: When you click "Reset to Draft" on a posted expense report, Odoo creates a Reversal Entry instead of setting the Journal Entry back to draft. This creates two locked entries for a single mistake.
  • Orphaned Entries: When you delete a draft expense report, the associated draft journal entries may remain in the system, causing confusion in the Journal Entries list.

The Solution

How this module fixes it

Smart Reset Logic

When you click Reset to Draft, this module intercepts the action. Instead of creating a reverse entry, it locates the original Journal Entry associated with the expense sheet and simply changes its status back to Draft.

Clean Deletion

When you Delete an expense sheet, the module automatically checks for any linked Journal Entries that are in a 'Draft' state and deletes them simultaneously. This ensures no orphaned data is left behind in your accounting journals.

Key Features

Revert to Draft

Moves the original Journal Entry back to draft state instead of creating a reversal.

Auto Cleanup

Automatically deletes associated draft journal entries when the expense sheet is deleted.

Cleaner Ledger

Keeps your General Ledger clean by avoiding unnecessary debit/credit cancellations.
