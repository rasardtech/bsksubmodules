Hide Decimal Zero Value

Features
  • Ability to hide decimal zero value from float and monetary fields in Odoo views and PDF Reports.
  • Working with any decimal point and thousand separator symbols.
    • How it works?

    You need only to install the module without any further configuration.

    • If the decimal value in float and monetary fields equal to zero, then it will be shown as integer in Odoo views and PDF reports. For example, If the value equal to 200.00 or 200.000 (depend on your field decimal points) then it will be shown as 200.
    • If the decimal value greater than zero for example 200.05 then it will be shown as its.
    • This app doesn't change the decimal accuracy and it will be shown as per digits in float fields and currency decimal point in monetary fields
    • This app work only on Odoo backend and PDF reports and not working on Odoo frontend like Ecommerce or POS
